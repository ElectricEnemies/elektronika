dimension module (3D effect)

directX 3d effects and video out

input/output

1 audio in
4 video in

properties

the dimension effect can handle 3 input effects followed by 3 display effects

the dimension effect control are on the right side of the interface
- power knob
- input effect selection
- effect selection
- blend mode
- color adjust

the default input is video. select the video source or background
the default effect is morph, use this to deform the image
