network 23 module (in/out)

send/receive audio/video stream using local network.

input/output

1 audio in
1 video in
1 audio out
1 video out

properties

browse network dialog to find network computer name
client/server, select client mode by clicking client button (default server)
select port number (both server and client should have the same port number)
power on/off button, start streaming
adjust audio master, bass, medium and treble

this module needs to be run on 2 computers. the stream is sent from from the server to the client
select one computer as the server (leave as default) and the other as the client (click the client button)
on the client enter the name of the server
start the server first and then the client
