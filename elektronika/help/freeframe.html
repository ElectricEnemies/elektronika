FreeFrame

FreeFrame is an open-source cross-platform real-time video effects plugin API. FreeFrame provides a framework for developing video effects, plugins and hosts for Windows, Linux and Mac OSX. Download a plugin and place it in elektronika's FreeFrame directory. It will then be accessible through this interface. Adjustable parameters vary with plugin.
