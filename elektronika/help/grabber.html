help file

 

grabber module (in/out)

capture video stream from a selectable desktop zone or window

input/output

1 video out

properties

select the desktop area or window to capture
overlay, turns off the overlay
overlay must be turned off before launching any software that uses an overlay window such as windows media player, real player etc
warning: if you use the overlay window as a source for the video out module don't use this button as lots of video cards only allow 1 overlay area
