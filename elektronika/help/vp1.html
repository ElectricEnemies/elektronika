VP1 player module (player)

media player with speed controls and looping

input/output

1 video out
1 audio out

properties

file pool. drag and drop video/mp3/wav/wma files onto the file pool
stop/play/pause. press button
seek bar. click and drag to select play point
auto play. auto start playing movie when selected in file pool
loop mode. restart movie from the beginning when ended
large dial. adjust play rate

Usage

clips should be the same size as your output window ie 320x240. it helps to have a keyframe every 3 frames and audio interleaved every frame. this speeds up loading and seek time. helps keep your fps up too.
a useful free converter/encoder is virtualdub
