web server module (in/out)

send video stream to the web (or web client module)

input/output

1 video in

properties

you must set the name or ip address of the computer and port (80 is the normal port for http)
the module uses the windows name of your computer by default for lan use
for internet use use a no dns service like no-ip.com or yi.org, to set your url
use the active button to activate the module
the number on the right is the quality of the stream using jpeg values 1-100

after running elektronika and activating the web server you can see a snapshot of the stream on your web browser by using the same url that you used in the web server module

html root is based in the elektronika/www directory. you can edit this page and create your own server. see the source html to get more information about the snapshot
