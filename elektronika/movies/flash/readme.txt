elektronika interact with the action script using somes vars:

elek_bass, elek_medium, elek_trebble: 	[0..32768]
elek_pad0...elek_pad5:  		[0..128]
elek_beat 				[0..infinite] number of beats from the start point
elek_text,text				set by elektronika edit text box
{ the text var is used to be compatible with resolume(tm) flash movies }


see elektronika help file (flash player) to get more informations
