dimension module
directX 3d effects and video out

inputs/outputs

1 audio in
1 video in
1 video out

usage

performance warning, if you plug a video stream on the video output this incur a high performance cost.
it's better to use this effect as video out.
with matrox video card just use the seclect a simple window device and use the zoom option of the video driver.
with others dual head video card use the corresponding directX device (second device).

the dimenson effect can instanciate three input effects followed by three display effects.

the dimension effect control only the right part of the interface
- power knob
- input effect selection
- effect selection
- blend mode
- color adjust

the default input is "video", select the video source or background
the default effect is morph, with it you can make somes deformation effect.
