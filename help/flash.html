help file

 

flash player module
play flash movie (SWF files)

inputs/outputs

1 audio in
1 video out

usage

select (1/8), select the movie
zoom, zoom the movie
colors , send variables to flash movie [elek_color0, elek_color1]
flash controls, send variables to flash movie [elek_pad0..elek_pad5]
text, edit text, sent as variable to flash movie [elek_text]

to load flash movies just drag'n drop your SWF files on the flash player rmodule.

the flash movie receive also the peak level for the 3 bands (bass, medium, treeble)
as variables [elek_bass, elek_medium, elek_treeble] and the time from elektronika
[elek_time] add 1000 unit/secondes (sequencer 120 bpm (set as default in elektronika live version)).

you can retreive the vars in flash at _level0 instance (ex: _level0.elek_bass)

you can send also information from flash to elektronika by using the command "fscommand"
in your actionscript. accepted commands:

fscommand("visible", "1111111"); // turn all flash controls in elektronika visible
fscommand("visible" ,"0000000"); // same but not visible
fscommand("visible" ,"111000"); // the first three visible

fscommand("tooltips0", "the tooltip"); // set the tooltips(x) of the pad(x) [x=0....5]
