network23 module
send/receive video/audio stream by local network.

inputs/outputs

1 video in
1 audio in
1 video out
1 audio out

usage

browse, find network computer name
client/server, select mode
on/off, start streaming

launch this module on two computer.

the stream go from the server to the client.

select one computer as server (nothing todo) and the other as client (press the client button).
connect input pins of the server module to others module as you want
and the output pin of the client by the same way.
on the server enter the server network name (nothing todo, the good one have been setted)
in the client enter the name of the server.
start the server and after the client.
