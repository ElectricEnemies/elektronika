webserver module
send video stream to the web (or webclient module)

inputs/outputs

1 video in

usage

you must set the name or the ip address of the computer and port (80 is the normal port for http protocol). the module set by default the windows name of your computer to use it via lan. for internet if you use nodns service like no-ip.com, yi.org, set your url.

use the active button to activate the module

the number on the right is the quality of the stream [1~100]

after running elektronika and activate the web server you can see snapshot of the stream on you web brower by using the same url that you use in the webserver module.

html root is based in the elektronika/www directory, you can edit this pages and create your own server, see the source html to get more information about getting the snapshot..
